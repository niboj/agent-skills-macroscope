## Synthèse du livrable

L'administrateur de base de données apporte au projet des compétences de gestion de bases de données. Il est responsable principalement d'optimiser la performance des bases de données pour le système. Il est responsable de la conception physique globale et de l'intégrité de la ou des bases de données pour le système. Dans le cas d'un **projet de livraison Agile** , il n'y aura pas qu'une seule étape dans laquelle toute la conception de la base de données est faite. C'est un processus itératif et, de cette manière, le rôle d'administrateur de base de données doit pouvoir s'adapter et s'accorder à ce processus.
