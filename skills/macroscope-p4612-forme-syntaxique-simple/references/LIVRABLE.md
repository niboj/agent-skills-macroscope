## Synthèse du livrable

Les formes syntaxiques simples sont une forme de représentation de la modélisation données-processus. Elles servent à décrire, selon une perspective dynamique, le modèle de traitements dans la vue de l'utilisateur. Il est possible d'écrire une version du déroulement des activités dans un style plus structuré qui permet de mieux visualiser les différentes options offertes. Les formes syntaxiques simples décrivent les activités elles-mêmes composées d'activités plus petites, qui peuvent être exécutées dans une séquence. L'exécution de ces activités peut être soumises des conditions ou il peut y avoir répétition de ces activités.
