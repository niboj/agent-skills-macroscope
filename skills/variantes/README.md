# Variantes de livrables Macroscope

Ce répertoire regroupe les skills générées qui représentent une variante de contenu pour un livrable déjà couvert par une skill canonique dans `skills/`.

Les variantes ne sont pas exposées comme skills actives par défaut, car elles sont placées sous `skills/variantes/`. Elles servent de matière de consolidation pour enrichir les références des skills canoniques sans multiplier les déclencheurs.

Consulter `INDEX.md` pour relier chaque variante à sa skill canonique.
