# Motifs d evaluation

Inclure les deux :

- Qualite de declenchement : doit declencher vs ne doit pas declencher
- Qualite de sortie : ce que la skill doit produire concretement

Types de cas utiles :

- Cas nominal
- Entrees incompletes
- Prompt ambigu
- Prompt voisin qui ne doit pas declencher
- Regression pour un echec passe connu
