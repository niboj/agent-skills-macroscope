# Références Macroscope

Ce dossier contient un corpus local de pages Macroscope converties en Markdown.

## Usage

- Utiliser `macroscope/livrables/` comme source de consultation pour la terminologie, les conventions de représentation et les descriptions de livrables.
- Citer les chemins consultés dans une skill ou dans une synthèse lorsque leur contenu influence la sortie.
- Ne pas modifier les fichiers sources du corpus lors de la création d'une skill, sauf demande explicite.

## Organisation

- `livrables/` : pages Markdown indexées par identifiant technique.

Le corpus est volontairement conservé comme source brute. Les synthèses, index thématiques et conventions réutilisables doivent être ajoutés dans `references/` ou dans le dossier `references/` d'une skill spécialisée.
