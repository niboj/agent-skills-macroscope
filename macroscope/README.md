# References Macroscope

Ce dossier contient un corpus local de pages Macroscope converties en Markdown.

## Usage

- Utiliser `macroscope/livrables/` comme source de consultation pour la terminologie, les conventions de representation et les descriptions de livrables.
- Citer les chemins consultes dans une skill ou dans une synthese lorsque leur contenu influence la sortie.
- Ne pas modifier les fichiers sources du corpus lors de la creation d une skill, sauf demande explicite.

## Organisation

- `livrables/` : pages Markdown indexees par identifiant technique.

Le corpus est volontairement conserve comme source brute. Les syntheses, index thematiques et conventions reutilisables doivent etre ajoutes dans `references/` ou dans le dossier `references/` d une skill specialisee.
