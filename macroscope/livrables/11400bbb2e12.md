# http://intradoc-cadrenormatif.itp.extra/DMRPFr/html/Fr_P_351_99_OPD.html

|  Les modèles de conception créés par le « Gang of Four » (GoF en abrégé) sont classés en trois catégories :  
  * **Créationnel** - modèles qui décrivent comment régler les problèmes de création et de configuration d'objets.
  * **Structurel** - modèles qui décrivent comment structurer les classes afin d'avoir le minimum de dépendance entre objets.
  * **Comportemental** - modèles qui augmentent la flexibilité en identifiant des patrons de communication entre objets. Ces modèles décrivent une structure de classes pour le comportement de l'application (pour répondre à un évènement par exemple).
