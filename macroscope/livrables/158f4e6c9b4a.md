####  Description
Une circonstance d'utilisation correspond à l'utilisation du système par une ou plusieurs clientèles dans un ou plusieurs environnements de travail donnés. Les circonstances d'utilisation peuvent être indiquées à l'aide d'une matrice dont les lignes contiennent les environnements de travail et, les colonnes, les clientèles. Il suffit alors d'indiquer les combinaisons pertinentes dans la matrice.
####  Extrants
Les livrables suivants peuvent être utilisés pour documenter les résultats de cette activité de technique. Les sections des livrables qui sont applicables sont montrées sous chaque livrable. Les livrables détaillés qui peuvent être utilisés au lieu des livrables succincts, ou en complément à ceux-ci, sont listés directement sous les livrables succincts concernés.
