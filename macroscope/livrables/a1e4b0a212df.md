####  Description
Certains éléments de notation, tels que les attributs, les opérations et les multiplicités, sont communs à plusieurs diagrammes UML. D'autres éléments, tels que les commentaires, les contraintes, les stéréotypes et les noms qualifiés, sont généraux et peuvent être spécifiés dans tous les diagrammes UML. Ces éléments sont décrits dans cette section.
