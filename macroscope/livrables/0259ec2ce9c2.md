####  Description
Cette activité de technique traduit les modèles de classe utilisateur, produits par la technique [P-OO-UO Analyse d'objet détaillée](http://intradoc-cadrenormatif.itp.extra/DMRPFr/html/Fr_P_179_60_39_TEC.html), en une version préliminaire des modèles de classe réalisateur. Elle produit les opérations et les attributs des classes réalisateur en fonction des services définis pour les classes utilisateur. La traduction initiale en modèles de classe réalisateur se fait par facette. Une révision initiale des responsabilités est effectuée afin de déterminer les spécifications d'interface qui seront supportées par les classes. Une version préliminaire du modèle de l'information persistante est établie à partir des modèles de classe réalisateur. Les classes et diagrammes réalisateur produits par cette technique sont utilisés par la technique [P-OO-DI Modélisation des interactions de niveau réalisateur](http://intradoc-cadrenormatif.itp.extra/DMRPFr/html/Fr_P_174_60_39_TEC.html).
