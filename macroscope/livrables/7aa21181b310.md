####  Description
Pour bien comprendre le comportement des classes présentant des changements d'état nombreux ou critiques, on résume les effets de toutes les opérations sur l'état de ces classes au moyen de diagrammes d'état. Cette tâche aide à découvrir les opérations ou les états absents dans les spécifications de classe et à identifier les opérations redondantes.
